## 🚀 Deploy to Vercel in 3 Steps

### Step 1 — Upload to GitHub

1. Go to [github.com](https://github.com) and create a free account
2. Click **"New repository"** → name it `dani-global-academy` → click **Create**
3. On your computer, open a terminal in this folder and run:

```bash
git init
git add .
git commit -m "Initial commit"
git branch -M main
git remote add origin https://github.com/YOUR_USERNAME/dani-global-academy.git
git push -u origin main
```

### Step 2 — Deploy on Vercel

1. Go to [vercel.com](https://vercel.com) → Sign up with GitHub (free)
2. Click **"Add New Project"**
3. Find `dani-global-academy` → click **Import**
4. Leave all settings as default → click **Deploy**
5. ✅ Your site is live at `dani-global-academy.vercel.app` in ~60 seconds!

### Step 3 — Add Custom Domain (Optional, ~$12/year)

1. Buy your domain at [namecheap.com](https://namecheap.com) (e.g. `daniglobalacademy.com`)
2. In Vercel → Your Project → **Settings** → **Domains**
3. Type your domain → click **Add**
4. Copy the DNS records Vercel gives you
5. Go to Namecheap → Manage Domain → **Advanced DNS** → paste the records
6. Wait 10–30 minutes → your site is live on your custom domain! 🎉

## 💻 Run Locally

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview
```

Open [http://localhost:5173](http://localhost:5173) in your browser.

## ✨ Features

- 🏠 **Homepage** — Hero search, categories, featured scholarships, expiring soon
- 🔍 **Browse** — Filter by country, degree, award type with live search
- 📄 **Detail Pages** — Full scholarship info with tabs and apply button
- 🔐 **Auth** — Login & Register (student / organization roles)
- 📊 **Student Dashboard** — Bookmarks, kanban application tracker, settings
- 🏛 **Org Dashboard** — Post and manage scholarships
- ✍️ **4-Step Form** — Create and publish scholarships
- 📱 **Fully Responsive** — Works on mobile, tablet, and desktop
